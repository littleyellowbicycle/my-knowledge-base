## 🚀 快速开始

给 AI Agent 发一条消息即可：

> "帮我看看这个链接并整理到知识库 https://..."
>
> "我收藏过关于 AI Agent 的东西吗？"
>
> "把知乎收藏夹 https://zhihu.com/collection/... 全部抓取并归档"

无需手动操作，Agent 会自动调用 `ingest_and_process` → `ask` 完成全流程。
